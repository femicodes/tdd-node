## TDD for Node.js auth system

A basic authentication system built with Express. It stores users in a Mongo database and generates JWT tokens upon login.

### Technologies
 1. Node.js
 1. Express.js
 1. Mocha
 1. Chai/chai-http
 1. MongoDB
